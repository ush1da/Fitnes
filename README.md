# FitLife Fitness Club

Сучасний веб-застосунок для фітнес-клубу, розроблений у межах лабораторної роботи №1.

## 📋 Огляд проєкту
Проєкт складається з трьох взаємопов'язаних сторінок, що відповідають вимогам **"Базового рівня"**
* **Головна:** Landing-сторінка з Hero-секцією, абонементами та інтерактивною картою.
* **Розклад та заняття:** Складна таблиця розкладу з використанням семантичних блоків `<thead>`, `<tbody>`, `<tfoot>` та вбудованим відео-туром.
* **Запис до клубу:** Розгалужена форма зворотного зв'язку з валідацією даних та групуванням полів.

## 🛠 Технології
**HTML5:** Семантична розмітка (`<header>`, `<nav>`, `<main>`, `<footer>`)[cite: 7].
* **CSS3:** Сучасний дизайн у стилі Glassmorphism, використання шрифту **Inter**, адаптивна верстка.
* **Валідація:** Регулярні вирази (RegEx) для перевірки номерів телефонів.
* **Lighthouse:** Оптимізація під SEO та Accessibility (90+ балів).

## 🚀 Основні характеристики (Базовий рівень)
* **Структура:** Створено 3 взаємопов'язані сторінки (Головна, Каталог, Контакти).
* **Семантична розмітка:** Використано обов'язкові HTML5-теги: `<header>`, `<nav>`, `<main>`, `<footer>`.
* **Прайс-лист:** Побудовано таблицю для відображення вартості послуг та розкладу[cite: 8].
* **Форма зворотного зв'язку:** Реалізовано інтерфейс збору даних з полями для імені, email та повідомлення.
* **Мультимедіа:** Додано тематичні зображення з обов'язковим атрибутом `alt` для покращення доступності.
