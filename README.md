# 🏋️‍♂️ FitLife Fitness Club — Web Application

Сучасний веб-застосунок для фітнес-клубу, розроблений у межах лабораторної роботи №1. Проєкт демонструє навички семантичної верстки, роботи з мультимедіа та проектування користувацьких інтерфейсів.

## 📋 Огляд проєкту
[cite_start]Проєкт складається з трьох взаємопов'язаних сторінок, що відповідають вимогам **"Просунутого рівня"**[cite: 12, 13]:
* [cite_start]**Головна:** Landing-сторінка з Hero-секцією, абонементами та інтерактивною картою[cite: 7].
* [cite_start]**Розклад та заняття:** Складна таблиця розкладу з використанням семантичних блоків `<thead>`, `<tbody>`, `<tfoot>` та вбудованим відео-туром[cite: 15, 23].
* [cite_start]**Запис до клубу:** Розгалужена форма зворотного зв'язку з валідацією даних та групуванням полів[cite: 17, 18, 20].

## 🛠 Технології
* [cite_start]**HTML5:** Семантична розмітка (`<header>`, `<nav>`, `<main>`, `<footer>`)[cite: 7].
* **CSS3:** Сучасний дизайн у стилі Glassmorphism, використання шрифту **Inter**, адаптивна верстка.
* [cite_start]**Валідація:** Регулярні вирази (RegEx) для перевірки номерів телефонів[cite: 20].
* [cite_start]**Lighthouse:** Оптимізація під SEO та Accessibility (90+ балів)[cite: 25, 26].

## 🚀 Основні характеристики (Продвинутий рівень)
* [cite_start]**SEO:** Налаштовані мета-дані, viewport та Open Graph теги[cite: 14].
* [cite_start]**Складна таблиця:** Реалізовано об'єднання комірок (`colspan`) для акційних пропозицій[cite: 16].
* [cite_start]**Форми:** Використання `<fieldset>`, `<legend>` та атрибутів `required` для забезпечення чистоти даних[cite: 18, 20].
* [cite_start]**Мультимедіа:** Адаптивне вбудовування відео з YouTube[cite: 23].
